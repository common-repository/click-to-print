This plugin is to add print button to your posts and pages.

== Description ==
Use [ctp_print] shortcode to add "Print" button to print contents of any page or post.

== Installation ==

1. Upload `click-to-print` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. add [ctp_print] shortcode to your pages/posts.

Note: Don't change "click-to-print" directory name
